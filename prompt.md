Build a complete, playable Flappy Bird clone using only HTML, CSS, and JavaScript (no build step required — it should run by opening the page in a browser).

Requirements:
- **Gameplay:** Classic Flappy Bird mechanics — a bird that flaps upward on input and falls under gravity, scrolling pipes with gaps to fly through, score increments per pipe passed, and collision detection (pipes, ground, ceiling) that ends the game.
- **Assets:** If you can generate images, generate your own art for the bird, pipes, and background. Otherwise, draw the graphics programmatically (e.g., on a canvas).
- **Sound:** Include sound effects for at least flapping, scoring, and collisions.
- **Pause/Resume:** The player can pause and resume the game at any time during play.
- **High scores:** Keep a leaderboard of the top ten scores. On game over, prompt the player for a name (or initials) and store it alongside their score. Display the leaderboard with names and scores.
- **Persistence:** The leaderboard must survive a page reload (e.g., via localStorage).

You have full freedom over the visual design, controls, file/folder structure, and any libraries or techniques you use. Make it polished and fun to play.

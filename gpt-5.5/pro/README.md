# Skyhop — Flappy Bird Clone

A complete, build-free Flappy Bird style browser game made with HTML, CSS, and JavaScript.

## Run

Open `index.html` in any modern browser. No install, no build step, and no network required.

## Controls

- Space / Arrow Up / W: flap
- Click / tap the canvas: flap
- P / Escape: pause or resume
- Pause button: pause or resume

## Features

- Classic gravity, flap impulse, scrolling pipes, scoring, and collision detection
- Programmatic canvas art for the bird, pipes, skyline, clouds, ground, and effects
- Generated Web Audio sound effects for flap, score, pause, and collision
- Pause/resume during play
- Local top-10 leaderboard with names and scores
- localStorage persistence across page reloads
- Responsive layout for desktop and mobile

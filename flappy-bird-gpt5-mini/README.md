# Flappy Bird — Mini Clone

Simple polished Flappy Bird clone built with HTML/CSS/JavaScript.

Files:
- index.html — main page
- css/style.css — styles
- js/game.js — game logic

How to run:
Open `index.html` in your browser (modern Chrome/Firefox/Safari). If audio doesn't play, click the canvas to resume the audio context.

Controls:
- Space key to flap
- Click or tap the canvas to flap

Features:
- Canvas-based rendering
- Smooth bird physics
- Procedural pipes and scoring
- WebAudio synthesized sounds
- Local high score stored in localStorage

Enjoy!
